## 📝 프로젝트 소개

본 프로젝트는 **PUBG: BATTLEGROUND**을 모작하여, 원작의 게임 플레이 스타일을 참고했으며  
데디케이트 서버를 이용해 멀티플레이 게임을 제작한 팀 프로젝트입니다.
게임 플레이에 대한 핵심 시스템을 직접 개발하였으며, Gameplay Ability System을 적극적으로 사용하였습니다.
